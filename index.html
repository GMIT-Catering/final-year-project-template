Welcome to our site.

This page was created so that you could easily see the various components of the GMIT Catering Food Ordering System. We hope that all your answers will be answered here.

Components

The various components of the system include:

  • Web App - for taking orders (staff)
  • Mobile App - for customers to order food and interact with the company
  • Push Web App - for staff to send push notifications to users

Web App

The web app is used by the staff to view and manage users, orders and to update information about the application.

Usage Instructions

Logging in:

  • Log in via: http://canteenapplication.vmarisevs.me/login

New User:

  • Add a new user through the account settings tab

Basic Usage

  • Click on each of the tabs
  • They are self explanatory

Mobile App

The mobile app is used by the customer to view information about the GMIT Catering Company, order food, and to receive alerts via push notifications.

Privacy Policy: http://www.ronanconnolly.ie/#/gmitcat-priv-pol

Usage Instructions

Installation:

  • Download the app for either Android(old version)(https://play.google.com/store/apps/details?id=ie.gmit.gmitcatering2015) or iOS(https://itunes.apple.com/ie/app/gmit-catering/id1054850061?mt=8).

Signing up:

  • Click on Sign Up
  • Make sure to enter unique/valid name, email and GMIT ID
  • You will be automatically logged in

Topping up:

  • Visit the GMIT Catering Company checkout to purchase a €10 top up card
  • Tap the top-up option in the settings tab
  • Scan the top up card
  • You will then see your account credit within the app updated

Ordering:

  • Tap on the Orders tab
  • Select a sandwich, additionals, a time and confirm
  • Visit the GMIT Catering Sandwich bar and collect your sandwich (you must show your GMIT ID)

Push Web App

The web app is used by the staff to view how many users are registered for push notifications, what push messages were previously sent and to send new push notifications.

Usage Instructions

Logging in:

  • Log in via: https://gmitcat-push.herokuapp.com/#/login
  • Username: gmitcat
  • Password: tactimg
  • These details will be changed

Sending a push notification:

  • Type message and click send
  • The message status will update once it has sent successfully
